Overview

Akyro is a Python/PyQt6 desktop application designed for intelligent document management. It converts PDFs and DOCX into a custom editable format (.akr), allowing advanced editing, merging, and OCR-based recognition. Its UI is built for simplicity, clarity, and performance — similar in feel to InDesign, but focused on speed and automation. The application was developed in Italian for convenience, as my native language is Italian.

Technical Highlights

  • Built in Python 3.11 with PyQt6 for the GUI.
  • Automatic PDF/DOCX import and conversion to .akr proprietary format.
  • Integrated OCR for scanned documents (Tesseract-based).
  • Internal database to store metadata and file associations.
  • Editable paragraph widgets with drag-and-drop images and layout control.
  • Secure architecture: logic separated from UI; no open-source code distributed.
