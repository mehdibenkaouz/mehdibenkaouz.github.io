2D Infinite Room Generator – Unity Game (C#)

Developed a procedural 2D game environment in Unity featuring an infinite sequence of dynamically generated rooms. Implemented a centralized Room Manager system to instantiate, recycle, and scale rooms in real time, including illusion-based transitions and variable zoom effects. Integrated smooth camera control, dynamic player scaling, and parameter-driven speed adjustments to ensure consistent gameplay feel across infinite levels. Optimized performance with coroutine-based transitions, adaptive scaling, and efficient memory management through active room lists and on-demand destruction. Technologies: Unity, C#, Coroutines, Prefabs, Procedural Generation, 2D Camera Systems, Object Pooling
