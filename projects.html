Akyro — Document Intelligence App

A Python desktop application for intelligent document management. You can rename document stacks, merge documents, and separate documents. You can also create documents from scratch. The application offers the ability to convert PDFs and DOCXs to a custom editable format of your own creation (.akr) with OCR, visual text editing, and layout tools inspired by Adobe InDesign. It includes drag-and-drop import, a file merging system, and a PyQt6-based graphical user interface designed for clarity and efficiency.
