Selected Technical Projects

Akyro — Document Intelligence App

A Python desktop application for intelligent document management. Converts PDFs and DOCX into a custom editable format (.akr) with OCR, visual text editing, and layout tools inspired by Adobe InDesign. Includes drag & drop import, file merge system, and a PyQt6-based GUI designed for clarity and efficiency.

Python PyQt6 OCR PDF GUI

Unity Exploration Game — 9D World

Experimental 3D sandbox game developed in Unity. The player explores surreal worlds and transitions across 9 dimensions — shifting from 3D to 2D planes and higher spaces. All models, textures, and ambient soundtrack are self-produced. Focused on physics, camera logic, and world immersion.
